- Minikube CI/CD Pipeline with Python Time App
  
- Project Overview:
This project demonstrates a complete CI/CD pipeline for a Kubernetes-based application using modern DevOps tools.
The application is a simple Python service that prints the current system time and is deployed using a cloud-native workflow.
The project covers:
Local Kubernetes deployment using Minikube
Containerization using Docker
Image storage in Docker Hub
Continuous Integration using GitHub Actions
Continuous Delivery using Helm and ArgoCD (GitOps)

- Application
A lightweight Python application that prints the current system time.

- Example output:
The current time: 2026-04-17 07:47:38
